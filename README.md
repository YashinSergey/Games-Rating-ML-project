# Board Games Rating Prediction

Проект по предсказанию рейтинга настольных игр на основе характеристик игры. Модель обучалась на данных BoardGameGeek и сравнивалась на Kaggle.

## Задача

Построить модель машинного обучения, которая предсказывает **средний рейтинг игры (Rating Average)** на основе её характеристик.  
Метрика соревнования Kaggle — **Mean Squared Error (MSE)**.

## Данные

Используются два датасета:

- `train_data.csv` — обучающая выборка  
- `test_data.csv` — тестовая выборка  

Основные признаки:

- Year Published
- Min Players
- Max Players
- Play Time
- Min Age
- Users Rated
- Complexity Average
- Mechanics
- Domains

Целевая переменная: `Rating Average`


## Предобработка данных

На этапе предобработки были выполнены следующие действия:

- удалены неинформативные признаки **ID** и **Name**
- удалены признаки **Owned Users** и **BGG Rank** из-за сильной корреляции
- исправлены некорректные значения в признаках **Min Players**, **Max Players**, **Play Time** и **Min Age**
- значения `Max Players = 0` и `Max Players > 20` заменены средними значениями по признаку **Mechanics**
- значения `Play Time = 0` заменены средними по **Mechanics**
- выбросы `Play Time > 600` ограничены значением **600**
- в признаке **Complexity Average** нулевые значения заменены медианой
- строки с десятичной запятой приведены к числовому формату
- признаки **Mechanics** и **Domains** очищены и приведены к единому формату

В результате датасет не содержит пропусков и некорректных значений.

## Feature Engineering

На этапе feature engineering:

- создан новый признак `Game Age`
- признак `Year Published` удалён
- категориальные признаки **Mechanics** и **Domains** закодированы методом **one-hot encoding**

В результате было получено более **200 признаков**.

## Обучение моделей

Были протестированы несколько моделей:

- Linear Regression
- KNN Regression
- Support Vector Regression (SVR)
- Decision Tree
- Random Forest

Оценка проводилась на **валидационной выборке**.

Лучшие результаты показали: `SVR`, `Random Forest`


После подбора гиперпараметров лучшей моделью стал **Random Forest**.

## Результаты Kaggle

Финальные модели были проверены на тестовых данных Kaggle.

| Модель | MSE |
|------|------|
| SVR | 0.568 |
| Random Forest | **0.452** |

Random Forest показал лучший результат и выбран в качестве финальной модели.
